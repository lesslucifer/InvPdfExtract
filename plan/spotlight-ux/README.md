# Epic: Spotlight UX

Overlay-centric UX redesign for InvoiceVault. Replaces the tray icon + context menu with a single Spotlight-style overlay that handles all interactions: vault init, search, browse, export, and settings.

## Phases

| Phase | Name | Description | Status |
|-------|------|-------------|--------|
| 1 | [Foundation](phase-1-foundation.md) | State machine, new types, IPC handlers, settings panel | Pending |
| 2 | [Home & Folders](phase-2-home-folders.md) | Home screen with recent/top folders, folder browsing | Pending |
| 3 | [Filter Pills & Breadcrumbs](phase-3-filters-breadcrumbs.md) | Clickable paths, filter pills, breadcrumb bar | Pending |
| 4 | [Aggregation & Export](phase-4-aggregation-export.md) | Sticky footer, filtered export, Escape cascade | Pending |
| 5 | [Tray Removal & Polish](phase-5-tray-removal-polish.md) | Remove tray, E2E tests, cross-platform QA | Done |
| 6 | [Path Search](phase-6-path-search.md) | `/` prefix triggers inline folder/file picker, replaces `in:` filter | Pending |
